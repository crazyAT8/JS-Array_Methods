JavaScript Array Method: .reverse()

    Definition:
  • this method reverses the order of the elements in an array.
  • this method overwites the original array.

Syntax

array.reverse()

Return Value
The array after it has been reversed.
    Notes:
  • reverse() is an ECMAScript1 (ES1) feature.
  • ES1 (JavaScript 1997) is fully supported in all browsers.
    Reference:
  • W3schools

Example

Code:

const fruits = ["Banana", "Orange", "Apple", "Mango"]; document.getElementById("example1").innerHTML = fruits.reverse();

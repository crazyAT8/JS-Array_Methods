JavaScript Array Property: constructor
        

        
            Definition and Usage
        
            - returns the funcion that created the Array prototype.
            - ES1 (1997)

        
            Syntax:
        
            array.constructor
        
            Return Value:
        
            function Array() {[native code]} 
        
Example:

            Code:
            const fruits = ["peach", "mango", "tangerine"];

            const text = fruits.constructor;

            document.getElementById("example1").innerHTML = text;

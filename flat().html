JavaScript Array Method: .flat()

    Definition:
  • the .flat() method concatenates sub-array elements.

Syntax

array.flat(depth)

Parameters
Parameter
depth Optional.
How deep a nested array should be flattened.
Default is 1.
Return Value
Type Description
An array The flattened array.
    Notes:
  • .flat() is supported in all modern browsers since January 2020
  • Similar to these Array Methods
    • .map()
    • .flatMap()
    • .filter()
    • .forEach()

Example:

Code:

const num1Array = [[1,2],[3,4],[5,6]];
document.getElementById("example1").innerHTML = num1Array;
const newArray = num1Array.flat();
document.getElementById("example2").innerHTML = newArray;

const num2Array = [1,2, [3,[4,5,6], 7], 8];
document.getElementById("example3").innerHTML = num2Array;
const new2Array = num2Array.flat(2);
document.getElementById("example4").innerHTML = new2Array;

JavaScript Array Method: fill()

    Definition and Usage
  • this method fills elements in an array with a value
  • it overwrites the original array
  • start and end positions can be specified
  • if not, all elements will be filled

Syntax:

array.fill(value, start, end)

Parameters
Parameter Description
value Required. The value to fill in.
start Optional. The start index(position). Default is 0.
end Optional. The stop index (position). Default is array length.
Return Value
Type Description
An array The filled array.

Example:

    Code:
  • const girls = ["Brittney", "Jenny", "Ursala", "Becky", "Erin"]
  • document.getElementById("example1")
    .innerHTML = girls;
  • document.getElementById("example2")
    .innerHTML = girls.fill("Betty");
  • document.getElementById("example3")
    .innerHTML = girls.fill("Kathy", 0, 1);

    Notes:
  • fill() is an ECMAScript6 (ES6) feature
  • ES6 is supported in all modern browsers
  • not supported in Internet Explorer 11 (or earlier).
  • does not take multiple values

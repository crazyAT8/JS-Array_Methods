JavaScript Array Method: .at()

    Definition:
  • returns an indexed element from an array.
  • returns the same as [].

Syntax

array.at(index)

Parameters
Parameter Description
index Optional.
The index (position) of the array element to be returned.
Default is 0.
-1 returns the last element.
Return Value
Type Description
An Element The element of the given position (index) in the array.
    Notes:
  • .at() is supported in all browsers since March 2022
  • The bracked notation [] returns an indexed element from an array.
  • An empty index like at() defaults to at(0).
  • An index of -1 returns the last element from the array.

Example:

Code:

const fruits = ["Banana", "Orange", "Apple", "Mango"];
let fruit2 = fruits.at(2);
let fruit1 = fruits[1];
let fruitDefault = fruits.at();
let lastFruit = fruits.at(-1);

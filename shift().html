JavaScript Array Method: .shift

    Definition:
  • this method removes the first item of an array.
  • this method changes the original array.
  • this method returns the shifted element.

Syntax

array.shift()

Parameters
NONE
Return Value
Type Description
A variable The removed item.
A string, a number, an array, or any other type allowed in an array.
    Notes:
  • shift() is an ECMAScript1 (ES1) feature.
  • ES1 (JavaScript 1997) is fully supported in all browsers
    Reference:
  • W3schools

Example

Code:

const fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.shift(); // gets rid of Banana

document.getElementById("example2").innerHTML = fruits.shift();
//gets rid of Orange
document.getElementById("example1").innerHTML = fruits;

Filter Array Method

        Definition:                                                   Similar: map()
                This method creates a new array                                forEach()
                    filled with elements that pass a test 
                        provided by a function.
                It does not execute 
                    the function for empty elements
                        and does not change the original array 

        Syntax

            array.filter(function(currentValue, index, arr), thisValue)

        Parameters
        

            
                

                    Parameter
                    Description
                

                

                    function()
                    
                        Required.
                        A function to run for each array element
                    
                

                

                    currentValue
                    
                        Required.
                        The value of the current element.
                    
                

                

                    index
                    
                        Optional.
                        The index of the current element.
                    
                

                

                    arr
                    
                        Optional.
                        The array of the current element.
                    
                

                

                    thisValue
                    
                        Optional. Default undefined 
                        A value no elements pass the test it returns an empty array.
                    
                

            
        


        Return Value
        

            
                

                    Type
                    Description
                

                

                    An Array
                    
                        Containing the elements that pass the test.
                        If no elments pass the test it returns an empty array.
                    
                

            
        


        Example:
                                    Click "Test" to get every element in the array that has a value above this number: 
                                    
                                    
                Return Value:

        Code:
        const numbers = [1, 24, 45, 73, 26, 38, 29, 17, 13, 11, 23];

        function checkNumber(numbers) {
            return numbers > document.getElementById("numberToCheck").value;
        }

        function philter2() {
            document.getElementById("test").innerHTML = numbers.filter(checkNumber);
        }

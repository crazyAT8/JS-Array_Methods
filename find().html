JavaScript Array Method: find()

    Definition:
  • returns the first value that passes a test.
  • it executes a function for each array element.
  • returns undefined if no element is found
  • does not execute the function for empty elements.
  • does not change the original array

Syntax:

array.find(function(currentValue, index, arr),thisValue)

Parameters
Parameter Description
function() Required.
A function to run for each array element.
currentValue Required.
The value of the current element.
index Optional.
The index of the current element.
arr Optional.
The array of the current element.
thisValue Optional. Default undefined.
A value passed to the function as its this value.
Return Value
Type Description
A value The value of the first element that pass the test. Otherwise it returns undefined.
    Notes:

Example:

Code: Ex.1

const ages = [3, 30, 18, 20, 25];
function checkAge(age) {
   return age > 18;
}
document.getElementById("example1").innerHTML = ages.find(checkAge);

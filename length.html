JavaScript Array Property: .length

    Description:
  • The length property sets or returns the number of elements in an array.

Syntax

Return the length of an array:

array.length

Set the length of an array:

array.length = number

Return Value
Type Description
A number The number of elements in the array.
    Notes:
  • length is an ECMAScript (ES1) feature.
  • ES1 (JavaScript 1997) is fully supported in all browsers
    Reference:
  • W3schools

Example

Code:

const fruits = ["Banana", "Orange", "Apple", "Mango"];
let length = fruits.length;
fruits.length = 2;

document.getElementById("example1").innerHTML = length;
document.getElementById("example2").innerHTML = fruits;

JavaScript Array Method: map()

    Definition:
  • creates a new array from calling a function for every array element.
  • does not execute the function for empty elements.
  • does not change the original array

Syntax

array.map(function(currentValue, index, arr), thisValue)

Parameters
Parameter Description
function() Required.
A function to be run for each array element.
currentValue Required
The value of the current element.
index Optional.
The index of the current element.
arr Optional.
The array of the current element.
thisValue Optional.
Default value undefined.
A value passed to the function to be used as its this value.
Return Value
Type Description
An array The result of a function for each array element.
    Notes:
  • map() is an ECMAScript5 (ES5)
  • ES5 (JavaScript 2009) fully supported in all browsers
  • Basically, you create a function that needs to be executed on every array item
          and that function is the parameter of map()

  • W3schools

Example

Code:

const numbers = [4, 9 ,16, 25];
const newArr = numbers.map(myFunction);
const persons = [
   {firstname : "Malcom", lastname: "Reynolds"},
   {firstname : "Kaylee", lastname: "Frye"},
   {firstname : "Jayne", lastname: "cobb"}
];

function myFunction(num) {
   return num * 10
}

function getFullName(item) {
   return [item.firstname,item.lastname] .join(" ");
}

document.getElementById( "example1").innerHTML = numbers.map(Math.sqrt);
document.getElementById( "example2").innerHTML = newArr;
document.getElementById( "example3").innerHTML = persons.map(getFullName);

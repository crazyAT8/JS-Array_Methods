JavaScript Array Method:

    Definition:
  • this method returns an Array Iterator object with the keys of an array.
  • this method does not change the original array.

Syntax

array.keys()

Parameters
NONE
Return Value
Type Description
An array An Array Iterator object containing the keys of an array.
    Notes:
  • is an ECMAScript6 (ES6) feature.
  • ES6 (JavaScript 2015) is supported in all modern browsers
    Reference:
  • W3schools

Example

Code:

const fruits = ["Banana", "Orange", "Apple", "Mango"];
const keys = fruits.keys();
const keys2 = Object.keys(fruits);
let text = "";
for (let x of keys) {
  text += x + "<br>"
}
let text2 = "";
for (let x of keys2) {
  text2 += x + "<br>"
}
document.getElementById("example1").innerHTML = text;
document.getElementById("example2").innerHTML = text2;

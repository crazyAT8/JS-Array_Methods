JavaScript Array Method: concat()
        

        
            Definition and Usage
        
            - concatenates (joins) two or more arrays
            - returns a new array, containing the joined arrays 
            - does not change the existing array 
            - ES1 (1997)

        
            Syntax:
        
            array1.concat(array2, array3, ..., arrayX)
        
            Parameters
        
            - array2 (required)
            - all others are optional
        
            Return Value
        
            type: an array
            description: the joined arrays 
        
Example:

            Code:
            const arr1 = ["peach", "mango", "tangerine"];
            const arr2 = ["lemon", "lime", "orange"];
            const arr3 = ["Banana"];

            const fruits = arr1.concat(arr2, arr3);

            document.getElementById("example1").innerHTML = fruits;

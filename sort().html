JavaScript Array Method: sort()

    Definition:
  • this method sorts the elements of an array.
  • this method overwrites the original array.
  • this method sorts the elements as strings in alphabetical and ascending order.

Syntax

array.sort(compareFunction)

Parameters
Parameter Description
compareFunction Optional.
A function that defines a sort order. The function should return a negative, zero, or positive value, depending on the arguments:
  • function(a, b){return a-b}
When sort() compares two values, it sends the values to the compare function, and sorts the values according to the returned (negative, zero, positive) value.
Return Value
The array with the items sorted.
    Notes:
  • sort() is an ECMAScript1 (ES1) feature.
  • ES1 (JavaScript 1997) is fully supported in all browsers
  • Sorting alphabetically works well for strings ("Apple" comes before "Banana").
  • But, sorting numbers can produce incorrect results.
  • "25" is bigger than "100", because "2" is bigger than "1".
  • You can fix this by providing a "compare function"

Example

Code:

const fruits = ["Banana", "Orange", "Apple", "Mango"];

document.getElementById("example1").innerHTML = fruits.sort();
fruits.reverse();
document.getElementById("example2").innerHTML = fruits;

const points = [40, 100, 1, 5, 25, 10];
points.sort(function(a, b){return a-b});
document.getElementById("example3").innerHTML = points;
document.getElementById("example4").innerHTML = points[0];
document.getElementById("example5").innerHTML = points[points.length-1];

points.sort(function(a, b){return b-a});
document.getElementById("example6").innerHTML = points;

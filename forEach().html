JavaScript Array Method: .forEach()

    Definition:
  • this method calls a function for each elemenrt in an arrray.
  • this method is not executed for empty elements.

Syntax

array.forEach(function(currentValue, index, arr), thisValue)

Parameters
Parameter Description
function() Required.
A function to run for each array element.
currentValue Required.
The value of the current element.
index Optional.
The index of the current element.
arr Optional.
The array of the current element.
thisValue Optional. Default undefined.
A value passed to the function as its this value.
Return Value
undefined
    Notes:
  • .forEach() is an ECMAScript5 (ES5) feature.
  • ES5 (JavaScript 2009) fully supported in all browsers

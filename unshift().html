JavaScript Array Method: .unshift()

    Definition:
  • this method adds new elemnts to the beginning of an array.
  • this method overwrites the original array.

Syntax

array.unshift(item1, item2, ..., itemX)

Parameters
Parameter Description
item1
item2
..
itemX 		The item(s) to add to the array. Minimum one item is required.
Return Value
Type Description
A number The new length of the array.
    Notes:
  • unshift() is an ECMAScript1 (ES1) feature.
  • ES1 (JavaScript 1997) is fully supported in all browsers
    Reference:
  • W3schools

Example

Code:

const fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.unshift("Lemmon", "Pineapple");

document.getElementById("example1").innerHTML = fruits;
document.getElementById("example2").innerHTML = fruits.unshift("Grapes");
document.getElementById("example3").innerHTML = fruits;
